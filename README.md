# roadbook_controller
Código para un ESP32 mini usado como mando para controlar una tablet a modo de Roadbook / DMD2, etc

# Compatibilidad
## Placa
Probado con un ESP 32 mini D1

## Librerías
- Software del board: esp32 by Espressif Systems v2.0.17
- Bounce2 v2.71
- ESP32_BLE_Keyboard https://github.com/T-vK/ESP32-BLE-Keyboard v0.3.2
